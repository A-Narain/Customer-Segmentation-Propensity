# Screenshots & Visual Outputs

This folder contains screenshots and visual outputs generated during the analysis and modeling phases of the project.

## Included Visuals
- Exploratory Data Analysis (EDA) plots
- Feature distributions
- Correlation heatmaps
- RFM analysis visuals
- Customer segmentation / clustering results
- Model performance charts (if applicable)

## Purpose
These images are stored separately to:
- Keep notebooks clean and readable
- Allow easy reference in README files
- Improve project presentation for reviewers and recruiters

## Usage
The images in this folder are referenced inside:
- Jupyter notebooks
- The main project README

Example:
```md
![RFM Distribution](Notebooks/screenshots/rfm_distribution.png)
